# Tic Tac Toe Game in C

## Description
A console-based Tic Tac Toe game developed in C.

## Features
- Two-player gameplay
- Win and draw detection
- Input validation
- Simple console interface

## Language
- C

## How to Run

```bash
gcc tic_tac_toe.c -o tic_tac_toe
./tic_tac_toe
```
